Dignity TranslitHelper
================

An easy to use helper for translate russian to latin letters.

Installation
============

```
composer require dignityinside/translithelper
composer install
```

Usage example
=============

```php
use Dignity\TranslitHelper;

TranslitHelper::translit('привет_мир');
```
